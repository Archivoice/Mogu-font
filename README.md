# Mogu-font
Handwritten Mandarin Chinese font set.

Contains roughly 10130 characters, handwritten in Krita, configured in Fontforge.
The release includes three formats: OTF, TTF, and WOFF
The font set mainly covers traditional Chinese, and while having coverage, is only limited for simplified Chinese.
